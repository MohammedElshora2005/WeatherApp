# WeatherApp - Modern Weather Forecast

A beautiful and feature-rich weather application that provides real-time weather data, 5-day forecasts, interactive maps, and a stunning user interface with dark/light mode support.

## Features

- **Current Weather** - Temperature, humidity, wind speed, pressure, visibility, and sunrise/sunset times
- **5-Day Forecast** - Daily weather predictions with average temperatures
- **Interactive Map** - Location display using Leaflet.js
- **Weather Statistics** - Average, min, max temperatures and average humidity
- **Favorite Cities** - Save and manage your favorite locations
- **Auto Refresh** - Automatic updates every 5 minutes
- **Unit Toggle** - Switch between Celsius and Fahrenheit
- **Dark/Light Mode** - Theme toggle with persistent preference
- **Current Location** - Get weather for your current position
- **Fully Responsive** - Works perfectly on all devices

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, Grid, Flexbox, Animations)
- JavaScript ES6+
- OpenWeatherMap API
- Leaflet.js (Maps)
- Font Awesome 6
- Google Fonts (Inter)
